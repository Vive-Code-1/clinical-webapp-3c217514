## লক্ষ্য

Jane App-এর সমস্ত মূল ফিচার + ফরাসি/ইংরেজি টগল + ভিজ্যুয়াল কাস্টমাইজেশন নিয়ে একটি সম্পূর্ণ ক্লিনিক ম্যানেজমেন্ট প্ল্যাটফর্ম। তিন ধরনের ব্যবহারকারী: একক প্র্যাকটিশনার, মাল্টি-প্র্যাকটিশনার ক্লিনিক, এবং ক্লায়েন্ট পোর্টাল।

## ব্যাকএন্ড: আপনার নিজস্ব Supabase

Lovable Cloud-এর বদলে আপনার নিজের Supabase প্রজেক্ট ব্যবহার করব। সংযোগ করার জন্য নিচের বোতাম চাপুন — এতে আপনাকে Supabase-এ লগইন করিয়ে প্রজেক্ট বেছে নিতে বলা হবে। আমার পক্ষ থেকে স্বয়ংক্রিয়ভাবে সংযোগ করার উপায় নেই, আপনাকেই Lovable UI থেকে যুক্ত করতে হবে।

<presentation-actions>

সংযুক্ত হলে Lovable স্বয়ংক্রিয়ভাবে `src/integrations/supabase/client.ts` তৈরি করবে এবং `VITE_SUPABASE_URL` / `VITE_SUPABASE_PUBLISHABLE_KEY` সেট করবে। তারপর আমরা মাইগ্রেশন চালিয়ে স্কিমা তৈরি করব।

## গুরুত্বপূর্ণ নোট

এটি একটি বিশাল প্রকল্প — Jane App-এর একটি দল কয়েক বছর ধরে তৈরি করেছে। একটি চ্যাটে সম্পূর্ণ তৈরি সম্ভব নয়। আমরা **৬টি ফেজে** ভাগ করে তৈরি করব। প্রতিটি ফেজ শেষ হলে আপনি পরীক্ষা করতে পারবেন এবং পরের ফেজে যাব।

## ফেজ ১: ভিত্তি, ডিজাইন সিস্টেম এবং অথেনটিকেশন

**লক্ষ্য:** সুন্দর ডিজাইন, FR/EN টগল, লগইন/সাইনআপ, ইউজার রোল।

- **আপনার Supabase সংযোগ যাচাই** — উপরের বোতাম থেকে সংযুক্ত হওয়ার পর।
- **ডিজাইন ডিরেকশন তৈরি** — সুন্দর আধুনিক হেলথকেয়ার-উপযোগী UI-এর ৩টি ভিন্ন ডিরেকশন, আপনি একটি বাছবেন।
- **ডিজাইন সিস্টেম** — সেমান্টিক টোকেন (oklch), কাস্টম ফন্ট, রঙ থিম `src/styles.css`-এ।
- **i18n সেটআপ** — `react-i18next` দিয়ে FR/EN টগল, ভাষা localStorage-এ সংরক্ষণ, সব টেক্সট translation key থেকে।
- **অথেনটিকেশন** — ইমেইল/পাসওয়ার্ড + Google সাইন-ইন (Google enable করতে Supabase ড্যাশবোর্ডে provider চালু করতে হবে — আমি গাইড করব)।
- **রোল সিস্টেম** — `user_roles` টেবিল, enum `app_role`: `owner | practitioner | receptionist | client`; `has_role()` সিকিউরিটি ডেফিনার ফাংশন।
- **প্রোফাইল টেবিল** — `profiles` (নাম, ছবি, ভাষা পছন্দ, টাইমজোন), সাইনআপে auto-create trigger।
- **প্রটেক্টেড রুট** — `_authenticated/` লেআউট, রোল-ভিত্তিক রিডাইরেক্ট (ক্লায়েন্ট → পোর্টাল, প্র্যাকটিশনার → ড্যাশবোর্ড)।
- **ল্যান্ডিং পেজ** — মার্কেটিং সাইট (Home, Features, Pricing, About, Contact) আলাদা রুটে।

## টেকনিক্যাল বিবরণ

- **স্ট্যাক:** TanStack Start v1 + React 19 + Tailwind v4 + shadcn/ui।
- **ব্যাকএন্ড:** আপনার নিজের Supabase প্রজেক্ট (Lovable Supabase Integration দিয়ে যুক্ত)।
- **i18n:** `react-i18next` + `i18next-browser-languagedetector`; `src/locales/en.json`, `src/locales/fr.json`।
- **সিকিউরিটি:** সব টেবিলে RLS; `has_role()` দিয়ে রোল চেক; ক্লায়েন্ট শুধু নিজের ডেটা, প্র্যাকটিশনার শুধু নিজের ক্লায়েন্ট, ওনার সব।
- **পেমেন্ট:** Stripe; **SMS:** Twilio; **ভিডিও:** Daily.co।
